## Quick Start

### Installation

1. Download `HackedServer.jar` and `HackedCore.jar` (dependency).  
2. (For 1.17+) Download `ProtocolLib`.  
3. Place the `.jar` files in your `/plugins/` folder.  
4. Restart your server.  
5. Follow the configuration guide in the docs.
